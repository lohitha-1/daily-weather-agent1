# Daily Weather Agent

This project is an automated weather agent built using n8n and the OpenWeatherMap API.

## Features

* Fetches current weather data
* Runs automatically using a schedule trigger
* Uses OpenWeatherMap API
* Can store data in Google Sheets or databases

## Tech Stack

* n8n
* OpenWeatherMap API
* JSON workflow automation

## How it Works

1. Schedule Trigger runs daily
2. API call fetches weather data
3. Data is processed and stored

## API Used

OpenWeatherMap Weather API
